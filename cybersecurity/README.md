# 🛡️ Siber Güvenlik (Cybersecurity) Roadmap

Bu dizin, siber güvenlik alanında uygulamalı (hands-on) deneyim ve laboratuvar başarısı gerektiren sertifikaları listelemektedir.

## 📌 Sertifikasyon Haritası

### 1. eJPT (eLearnSecurity Junior Penetration Tester)
Sektöre giriş yapmak isteyenler için temel sızma testi yetkinliğini kanıtlayan başlangıç sertifikası.
* **Detay:** Nmap, metasploit ve burp suite gibi araçlarla ağ üzerinde temel sızma pratiklerini test eder. 48 saatlik VPN lab deneyimidir.

### 2. OSCP (Offensive Security Certified Professional)
Red Team ve ofansif güvenlik profesyonelleri için yaygın olarak kabul gören sektör standardı.
* **Detay:** 24 saat içinde makineleri ele geçirmeniz ve profesyonel bir sızma testi raporu sunmanız beklenir. Privilege Escalation (Hak Yükseltme) konularını kapsar.

### 3. OSEP (Offensive Security Experienced Penetration Tester)
İleri seviye penetrasyon testi sertifikası. AV/EDR bypass ve Active Directory güvenlik değerlendirmelerine odaklanır.
* **Detay:** 48 saatlik sınav. Özelleştirilmiş payload geliştirme (C#/PowerShell) ve evasion (atlatma) teknikleri içerir.

### 4. CISSP (Certified Information Systems Security Professional)
Yönetim, mimari ve bilgi güvenliği liderliği sertifikası.
* **Detay:** Güvenliğin genel mimarisini ve risk yönetimini değerlendiren endüstri standardı belgedir.

---

## 🛠️ Eğitim & Laboratuvar Ekosistemi

Sertifika hazırlığı için aşağıdaki platformlar önerilmektedir:

- **HackTheBox (HTB):** İleri seviye pratik amaçlı sunulan laboratuvar (Pro Labs) altyapıları.
- **TryHackMe (THM):** Temel kavramların ve Active Directory sömürü senaryolarının öğrenilebileceği eğitim yolları.
- **PortSwigger Web Security Academy:** Web uygulama güvenliği ve "Burp Suite" kullanımı için kapsamlı eğitim platformu.
- **Kişisel Homelab:** Proxmox, VMware vb. hypervisor'ler üzerinden kurgulanan kişisel test ortamları (Active Directory, Windows Server).
